MsPiggit by antis81

A Qt based, platform independent git user interface using libqgit2. Currently only a viewer!

The goal of MsPiggit is to become a platform independent, easy to use Git user interface with special focus on submodules. In the current, very early development state it provides a viewer to repositories with few features. Contributions are very welcome! Have a look at the GitHub issues.
